# Active Directory TryHackMe Homelab

## Overview
This project documents my hands-on Active Directory practice through TryHackMe labs.

The goal of this homelab is to build practical understanding of how Active Directory works in enterprise Windows environments and how IT and cybersecurity teams use it to manage identity, access, and authentication.
