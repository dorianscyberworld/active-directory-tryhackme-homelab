# Lab 005 – Security Findings Summary

## Objective
Summarize key security lessons learned from the Active Directory homelab.

---

## Security Findings
- Weak passwords increase the risk of account compromise.
- Excessive permissions can expose sensitive resources.
- Failed login attempts should be monitored closely.
- Account lockouts may indicate user error or suspicious activity.
- Poor access control can increase organizational risk.

---

## Recommendations
- Enforce strong password policies
- Enable multi-factor authentication (MFA)
- Review group memberships regularly
- Monitor failed login attempts
- Limit privileged access
- Apply least privilege principles

---

## Why This Matters
Security findings help organizations improve their defensive posture and reduce risk.

These concepts are critical for:
- Help Desk
- IT Support
- System Administration
- SOC Analysis
- Cybersecurity Operations

---

## Key Takeaway
This Active Directory homelab strengthened my understanding of identity management, authentication, access control, and enterprise security best practices.
