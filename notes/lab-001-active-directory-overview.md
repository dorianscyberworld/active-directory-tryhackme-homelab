# Lab 001 – Active Directory Overview

## Objective
Understand what Active Directory is and why organizations use it.

---

## What I Learned
Active Directory is Microsoft's directory service used to centrally manage users, computers, permissions, and authentication inside Windows environments.

It allows IT teams to control access to systems, resources, and company data from a central location.

---

## Key Concepts
- Domain Controller
- Domain Users
- Organizational Units (OUs)
- Security Groups
- Authentication
- Authorization

---

## Why This Matters
Active Directory is critical in enterprise IT environments because it manages identity and access across the organization.

Help desk teams use Active Directory daily for:
- Password resets
- Account unlocks
- User creation
- Group management
- Access requests

Cybersecurity teams use Active Directory for:
- Investigating suspicious logins
- Monitoring account activity
- Managing privileged access
- Detecting security risks
