# Lab 003 – Domain Enumeration

## Objective
Understand domain enumeration and why it is important in Active Directory environments.

---

## What I Learned
Domain enumeration is the process of identifying important information inside an Active Directory environment such as users, groups, computers, permissions, and domain structure.

This information helps IT teams understand the environment and helps security teams identify risks.

---

## Key Concepts
- Domain Discovery
- User Enumeration
- Group Enumeration
- Computer Enumeration
- Permission Review

---

## Why This Matters
Enumeration provides visibility into how an environment is structured.

Help desk teams benefit from understanding:
- User accounts
- Group memberships
- Computer assets
- Access permissions

Cybersecurity teams use enumeration for:
- Security investigations
- Attack surface awareness
- Misconfiguration detection
- Risk identification
