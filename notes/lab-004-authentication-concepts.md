# Lab 004 – Authentication Concepts

## Objective
Understand authentication concepts used in Active Directory environments.

---

## What I Learned
Authentication verifies a user’s identity before allowing access to systems, applications, or resources.

Active Directory uses authentication to ensure only authorized users can access company resources.

---

## Key Concepts
- User Login
- Password Authentication
- Kerberos Basics
- Failed Login Attempts
- Account Lockout
- Multi-Factor Authentication (MFA)

---

## Why This Matters
Authentication is one of the most important security controls in enterprise environments.

Help desk teams commonly troubleshoot:
- Password reset requests
- Account lockouts
- MFA setup issues
- Failed login attempts

Cybersecurity teams investigate:
- Suspicious login activity
- Brute force attempts
- Unauthorized access attempts
- Account compromise risks
